# 小飞侠

小飞侠是一个基于 `Electron + React + TypeScript` 的飞书聊天记录导出工具。

项目目前处于初始阶段，只能完成一部分基础能力和简单需求，还不属于完整、稳定、覆盖全面的正式产品。

当前项目主要用于：

- 创建并接入飞书应用
- 使用飞书账号完成授权登录
- 浏览联系人、会话和消息内容
- 导出当前会话为 ChatLab `0.0.2` 标准格式
- 支持导出 `JSON` 和 `JSONL`

## 当前能力

当前只覆盖基础场景，适合验证流程和做早期使用：

- 联系人页区分为“官方通讯录”和“会话发现”
- 会话页支持群聊和 `P2P` 私聊
- 消息页支持用户侧消息和部分 Bot 会话补充
- 当前会话可直接导出为 ChatLab 格式

## 技术栈

- Electron
- React 19
- TypeScript
- Vite
- MUI
- 飞书开放平台 Node SDK

## 本地开发

```bash
npm install
npm run dev
```

## 目录说明

- `src/`：前端页面和界面逻辑
- `electron/`：主进程、IPC、飞书接口调用、数据存储
- `resources/`：图标和静态资源
- `scripts/`：开发辅助脚本

## 导出说明

消息页右上角提供“导出 ChatLab”按钮。

当前导出内容包括：

- `chatlab`
- `meta`
- `members`
- `messages`

当前导出限制：

- 头像默认导出为网络 URL，不是 Data URL
- 成员角色、群昵称等字段暂未完整补齐
- 外部联系人无法通过官方接口全量枚举，只能导出当前可见会话中的数据

## 注意事项

- 项目仍处于初始阶段，很多高级能力、边界场景和异常处理还没有完全补齐
- 当前实现更偏向“能跑通基础链路”，不保证覆盖所有飞书数据场景
- 修改权限范围后，通常需要重新授权登录一次
- Bot 会话是否完整可见，取决于飞书后台权限和事件订阅配置
- 外部联系人显示依赖“官方通讯录”或“已出现在可见会话中”这两条链路
